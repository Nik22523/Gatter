# Logikgatter

Ein kleines Java-Projekt zur Simulation von logischen Gattern.

## Beschreibung
Dieses Programm modelliert einfache digitale Logik mithilfe von objektorientierter Programmierung.  
Gatter können kombiniert werden, um komplexe logische Ausdrücke zu berechnen.

## Features
- UND, ODER und NICHT Gatter
- Eingänge mit WAHR / FALSCH
- Gatter kombinierbar (verschachtelt)
- Ausgabe als Boolean und Text

## Starten

javac *.java
java Test
